How It Works

A two-stage deep learning pipeline extracts spatial and temporal features from video.

01
Frame Extraction
OpenCV uniformly samples 16 frames from your video, regardless of duration.
02
CNN Feature Extraction
Each 224×224 frame passes through frozen ResNet50, producing 2,048 spatial features.
03
LSTM Temporal Modeling
A 2-layer LSTM reads the (16 × 2048) sequence to capture motion patterns over time.
04
Action Classification
A softmax output layer assigns probabilities across 20 action classes and returns the top prediction.
Model

Model Specifications

Trained on UCF-101 action dataset with 20 carefully selected activity classes.

Architecture
  • Feature Extractor ResNet50 (ImageNet)
  • Temporal Model 2-Layer LSTM
  • Frames per Video 16 uniform
  • Input Resolution 224 × 224 × 3
  • CNN Output Shape (16, 2048)
  • LSTM Units 256 → 128
  • Total Parameters 2,567,508
  • Output Classes 20 (softmax)
